Introduction

The data analysis module of the project "DIGITAL COMMUNICATION STRATEGIES FOR THE CULTURAL SECTOR IN THE BODENSEE REGION" dealt with the question of how digitisation was actually implemented using the example of some cultural providers on Lake Constance. For this purpose, selected methods of data mining and machine learning were used to analyse samples from the Internet. In this report interesting places are marked in italics and special conclusions are underlined.
